# Державний реєстр реєстраторів розрахункових операцій (ДРРРО)

## Мета проекту
Мета: дослідити та проаналізувати відкритий датасет про моделі реєстраторів розрахункових операцій (РРО), їх модифікації та виробників. Проект дозволяє:
- перевіряти якість та структуру даних,
- виявляти закономірності у строках реєстрації та термінах експлуатації,
- будувати візуалізації та аналітичні звіти для підтримки управлінських рішень у сфері торгівлі, громадського харчування та послуг.

## Джерело даних
Дані отримані з офіційного порталу відкритих даних України:
[Державний реєстр реєстраторів розрахункових операцій](https://data.gov.ua/dataset/8733e5b6-a46b-4c1d-bc94-bae88e83c859)

## Гіпотези для аналізу
1. **Гіпотеза 1:** Середній термін між первинною реєстрацією та виведенням з експлуатації відрізняється для вітчизняних та іноземних моделей.
2. **Гіпотеза 2:** Середня кількість рішень про внесення змін до реєстру суттєво відрізняється між лідерами ринку (ТОП-5 заявників), що вказує на різну активність підтримки обладнання.
3. **Гіпотеза 3:** Ринок РРО характеризується високою концентрацією виробництва, де майже половина доступних моделей належить трьом найбільшим заводам.
